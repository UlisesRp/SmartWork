SMART WORK CAPITAL HUMANO

Página web informativa para Smart Work Capital Humano, enfocada en servicios de reclutamiento, selección de personal, REPSE, nómina y bolsa de empleo.

ARCHIVOS PRINCIPALES

- index.html
- nosotros.html
- servicios.html
- repse-nomina.html
- bolsa-empleo.html
- contacto.html
- styles.css
- app.js
- assets/

FUNCIONES

- Sitio informativo responsive.
- Menú de navegación entre secciones.
- Botón flotante de WhatsApp.
- Formulario de contacto para cotizaciones y envío de CV.
- Sección de bolsa de empleo.
- Páginas separadas para Reclutamiento, REPSE y Nómina.
- Redes sociales agregadas como iconos.
